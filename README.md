# castle-client

This is the frontend for the castle server web application.

# Tests
Except the unite tests, which are implemented with Vitest,<br>
all other tests are created in Playwright.

### Unit  tests
To run these use the following command:
```
npm run test
```

### API tests
```
npx playwright test api
```

### E2E tests
```
npx playwright test e2e
```
